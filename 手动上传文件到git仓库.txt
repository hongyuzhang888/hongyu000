完整上传步骤（复制粘贴即可）
打开你要上传的文件夹，右键 → Git Bash Here（Windows），或直接终端进入文件夹（Mac/Linux）。
1. 初始化 Git 仓库
git init
2. 把文件添加到暂存区
git add .
只上传单个文件用：
git add 文件名.后缀
3. 提交文件（必须写备注）
git commit -m "第一次上传文件"
双引号里可以随便写，用来记录这次更新。
4. 关联你的 GitHub 仓库
git remote add origin https://github.com/你的用户名/你的仓库名.git
5. 推送文件到 GitHub（最终上传）
git push -u origin main
✅ 执行完这一步，文件就真正传到 GitHub 了！
